IBM Test Accelerator for Z

Fase 1, Sección 1: Visión Estratégica y Arquitectura

1.1. Definiendo la Propuesta de Valor

La adopción de IBM Test Accelerator for Z representa un cambio estratégico en el ciclo de vida de desarrollo de software (SDLC) para aplicaciones de mainframe. Su propósito fundamental es abordar los desafíos históricos asociados con las pruebas en z/OS. La propuesta de valor se centra en tres pilares clave:

  • Aceleración del Desarrollo (Shift-Left Testing): Permite a los desarrolladores crear y ejecutar pruebas unitarias y de integración temprana directamente desde su IDE.
  • Automatización Integral: La capacidad de grabar interacciones y generar automáticamente scripts de prueba reduce drásticamente el esfuerzo manual.
  • Eliminación de Dependencias: Mediante la virtualización de llamadas a subsistemas, las pruebas pueden ejecutarse sin necesidad de que el middleware o las aplicaciones dependientes estén disponibles.

1.2. Análisis Profundo de la Arquitectura

IBM Test Accelerator for Z es una suite de cuatro componentes interrelacionados:

  1. Dynamic Test Engine for z/OS: El corazón tecnológico del producto. Un motor de grabación y reproducción que se ejecuta en el host z/OS.
  2. Early Development Testing for z/OS (EDT): Herramienta orientada al desarrollador, integrada en un IDE basado en Eclipse como IDz.
  3. Integration Test Builder for z/OS (ITB): Herramienta para desarrolladores y testers, integrada en Visual Studio Code. Acelera la creación de pruebas de integración automatizadas basadas en Galasa.
  4. On-demand Dev and Test environment for z/OS (ODTz): Permite la creación y gestión de instancias virtuales de z/OS en hardware emulado.

1.3. Resolviendo la Confusión Central: El Rol de z/OS Debugger

La funcionalidad de grabación y reproducción no es un producto separado, sino una capacidad avanzada del depurador que se "desbloquea" con la licencia de Test Accelerator for Z. La habilitación del producto se realiza en el miembro IFAPRDxx de SYS1.PARMLIB.

  • Administrador de Seguridad (RACF)
  • Administrador de Red
  • Administrador de CICS/IMS
